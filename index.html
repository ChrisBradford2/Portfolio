Serveur Web

Ceci est mon serveur web. Il s'agit d'une Raspberry Pi 4 Model B 4Go avec l'OS Ubuntu Server.

Pour mettre en place mon serveur web, j'ai du installer Apache2 et modifier sa configuration afin de sécuriser mon serveur. J'ai également installé PHP en tant que processus FastCGI (qui est une manière plus récente d’installer PHP et souvent considérée comme plus performante). Enfin, j'ai ajouté une base de donnée avec MySQL.

Afin de tester mon serveur, j'ai décidé d'installer Wordpress. J'ai activé le site dans la configuration d'Apache et j'ai rentré l'adresse IP de mon serveur dans mon navigateur sur mon PC portable. Suite à ça, je suis tombé sur la page de configuration de Wordpress. J'ai rentré mon User, ma base de donnée, ainsi que mon mot de passe.

Ma page d'acceuil Wordpress s'est affichée, mon serveur web est fonctionnel !

Mon serveur étant accessible localement, j'aimerais le rendre accessible depuis Internet. Il me manques cependant des connaissances ainsi qu'un nom de domaine. Mais je continue au fil de mes nouvelles connaissances à améliorer mon serveur et, d'ici quelques mois, le rendre accessible partout !

Portfolio

Vous le consultez en ce moment même :)

L'idée de ce site m'est venu lorsqu'en faisant mon CV, j'inscrivait mes compétences, sans pour autant les démontrer. J'ai cherché des sites spécialisé pour faire son propre Portfolio comme Adoble Portfolio, Mahara ou encore Wordpress, mais aucun ne m'as véritablement convaicu. J'ai donc pris l'initiative de créer mon propre site web qui fera office de Portfolio.

Pour gagner en rapidité et partir sur une base, j'ai choisi le framework Bootstrap avec sa template Freelancer. J'ai modifié le code avec Adobe Dreamweaver afin de créer le site de mes envies.

Il est écrit en HTML, CSS et Javascript, avec une partie PHP pour le formulaire d'envoie du mail (que vous trouverez une section plus bas).
