Web Server

This is my web server. This is a Raspberry Pi 4 Model B 4GB with the Ubuntu Server OS.

To set up my web server, I had to install Apache2 and modify its configuration in order to secure my server. I also installed PHP as a FastCGI process (which is a newer way of installing PHP and often considered more efficient). Finally, I added a database with MySQL.

In order to test my server, I decided to install Wordpress. I activated the website in the Apache configuration and I entered the IP address of my server in my browser on my laptop. Following that, I came across the Wordpress configuration page. I entered my User, my database, and my password.

My Wordpress home page was displayed, my web server is functional!

My server being accessible locally, I would like to make it accessible from the Internet. However, I lack knowledge and a domain name. But over the course of my new knowledge, I continue to improve my server and, within a few months, make it accessible everywhere!

Snake in Javascript

Click on the image and press the space bar to start game.
Z = up; S = down; Q = left; D = right

While looking for a simple and fun project while allowing me to develop my skills, the Snake game appeared several times in several different object languages (many are in Python). Since I have great difficulty with object languages, I was advised to do so in Javascript. Which is ideal in my web development learning.

The game takes place in a canva to delimit the space of the game, the snake moves with the directional arrows and the fruits appear randomly in the canva without leaving it. The score is at the bottom right and the Game Over when lost is displayed at the top right. Just press Space at any point in the game to restart the game.

I would like to improve my game, first by making it more visually pleasing (by putting a background color and improving the score and Game Over display), but also by adding a pause system, a save system , and even a social media sharing score.
