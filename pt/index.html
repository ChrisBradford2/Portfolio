Quem sou eu ?

Servidor Web

Este é meu servidor web. Este é um Raspberry Pi 4 Modelo B 4GB com Ubuntu Server OS.

Para configurar meu servidor web, tive que instalar o Apache2 e modificar sua configuração para proteger meu servidor. Também instalei o PHP como um processo FastCGI (que é uma maneira mais recente de instalar o PHP e geralmente considerada mais eficiente). Finalmente, adicionei um banco de dados com MySQL.

Para testar meu servidor, decidi instalar o Wordpress. Ativei o site na configuração do Apache e digitei o endereço IP do meu servidor no navegador do meu laptop. Depois disso, me deparei com a página de configuração do Wordpress. Eu inseri meu usuário, meu banco de dados e minha senha.

Minha página inicial do Wordpress foi exibida, meu servidor da web está funcionando!

Sendo meu servidor acessível localmente, gostaria de torná-lo acessível pela Internet. No entanto, não tenho conhecimento e um nome de domínio. Mas, ao longo dos meus novos conhecimentos, continuo a melhorar o meu servidor e, dentro de alguns meses, a torná-lo acessível em todo o lado!

Snake en Javascript

Para jogar, clique na imagem e pressione a barra de espaço.
Z = subir; S = descer; Q = vá para esquerda; D = Vire à direita

Enquanto procurava um projeto simples e divertido que me permitia desenvolver minhas habilidades, o jogo "cobra" apareceu várias vezes em várias linguagens de objetos diferentes (notadamente em Python). Tendo grande dificuldade com linguagens de objetos, fui aconselhado a fazê-lo em Javascript. O que é ideal para meu aprendizado de desenvolvimento web.

O jogo acontece em uma canva para delimitar o espaço do jogo, a cobra se move com as setas direcionais e os frutos aparecem aleatoriamente na canva sem sair dela. A pontuação está no canto inferior direito e "Game Over" quando perdido é exibido no canto superior direito. Basta pressionar a barra de espaço em qualquer ponto do jogo para reiniciá-lo.

Gostaria de melhorar muito meu jogo, primeiro tornando-o mais bonito (colocando uma cor de fundo e melhorando a pontuação e o pôster Game Over), mas também adicionando um sistema de pausa, um sistema de salvar e até mesmo um sistema de compartilhamento de mídia social .
